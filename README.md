# 🤖 Gemini AI Chatbot

A smart conversational assistant powered by **Google Gemini API** and built using **Chainlit**. This chatbot is capable of understanding natural language, responding intelligently, and ensuring safety using built-in filters and fallback logic.

---

## 🚀 Demo Preview

> 💬 Ask me anything:  
> - “Who is the founder of OpenAI?”  
> - “Give me a summary of artificial intelligence.”  
> - “Tell me a fun fact about space!”

---

## ✨ Features

- ✅ Powered by **Gemini Flash** models
- 🔒 Integrated **rate limiting** (per minute & per day)
- 🔁 **Retry with exponential backoff** on errors
- 🤖 **Model fallback** if the default model fails
- 🛡️ **Safety filters** to avoid hate, harassment, violence, etc.
- ⚡ Built using **Chainlit** for interactive chat UI
- 🧠 Easily customizable and extendable

---

## 🧠 Technologies Used

- `Python 3.10+`
- `Chainlit`
- `Google Generative AI SDK`
- `dotenv`
